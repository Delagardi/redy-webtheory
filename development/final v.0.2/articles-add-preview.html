И снова всем привет. Я продолжаю свою серию текстовых туториалов по изучению языка Ruby. Сегодня у нас в планах:

  • - Установить Ruby интерпретатор
  • - Написать простенькую программу

Начнем мы, естественно, с первого пункта. Как вы помните, основным сайтом ruby-разработчика является http://ruby-lang.org/. Что ж, сегодня он нам понадобится.

Переходим на данный сайт, нажимаем на кнопочку Downloads (Скачать). На открывшейся странице мы можем увидеть 3 разных подхода для инсталяции интерпретатора:

  • - Компиляция исходников
  • - Использование сторонних инструментов
  • - Установка через системы управления пакетами

Windows

Для установки на операционных системах Windows нам понадобится утилита RubyInstaller. Официальный сайт- http://rubyinstaller.org/. Заходим на данный сайт, затем в раздел Downloads. Если вы знаете английский, то сможете понять, что нам рекомендуют установить Ruby версии 1.9.3. Устанавливаем данную версию. (У меня установлена именно эта версия).

UNIX-like OS

Для установки на unix-like ОС мы можем установить RVM (Ruby Version Manager). Команда: 
          
          ?php
          function just_do_it() {
          echo 'Test';
          }
Подведем итог:

  • - Сегодня мы установили интерпретатор Ruby
  • - Написали хеллоу ворлд
  • - Вскользь и тайно пробежались по следующему уроку (только цсс).

Задание #1

Напишите стандартную таблицу умножения от 1 до 10. Используйте циклы, манипуляторы. Совет: циклы можно вкладывать друг в друга как и операторы, используйте. Узнать решение
